O Sistema JAM será hospedado gratuitamente no Railway com todas as funcionalidades!

## 📋 **Passos para Deploy**

### **1. Instalar Railway CLI**
```bash
npm install -g @railway/cli
```

### **2. Fazer Login no Railway**
```bash
railway login
```

### **3. Criar Projeto**
```bash
railway init
```

### **4. Fazer Deploy**
```bash
railway up
```
